Project Description :
A responsive Employee Management System built with React and Vite to manage employee records, featuring a dashboard, employee listing, and employee registration with a clean and user-friendly interface.

 Project structure :
 Employee-management-system/
│
├── public/                      # Static assets
│
├── src/                         # Source code
│   ├── assets/                  # Images, icons, and other assets
│   ├── App.jsx                  # Main application component
│   ├── main.jsx                 # Application entry point
│   ├── Home.jsx                 # Home page component
│   ├── Dashboard.jsx            # Dashboard displaying employee statistics
│   ├── AddEmployee.jsx          # Form to add new employees
│   ├── EmployeeList.jsx         # Displays the list of employees
│   ├── Home.css                 # Styles for Home component
│   ├── Dashboard.css            # Styles for Dashboard component
│   ├── AddEmployee.css          # Styles for AddEmployee component
│   ├── EmployeeList.css         # Styles for EmployeeList component
│   └── index.css                # Global styles
│
├── package.json                 # Project dependencies and scripts
├── package-lock.json            # Dependency lock file
├── vite.config.js               # Vite configuration
├── eslint.config.js             # ESLint configuration

Installation and Setup :

Follow these steps to run the project :

Clone the Repository

<>bash
git clone https://github.com/Pawanpraneeth-stack/Employee-management-system.git


Navigate to the Project Directory

<>bash
cd Employee-management-system

Install Dependencies

<>bash
npm install

Start the Development Server

<>bash
npm run dev
